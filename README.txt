Create Schedule at URL: https://cs3733-scheduler.s3-external-1.amazonaws.com/static/scheduler.html

It will redirect you to the schedule view page after the schedule is created. That page URL (including the hash at the end) can be shared with participants. 

Ex: https://cs3733-scheduler.s3-external-1.amazonaws.com/static/schedule_view.html#72

To close a time slot, put the secret code into the text input field in the top right labeled "Secret Code", then press "close" on the appropriate slot. 

Participant-2 will not be able to schedule a meeting in Participant-1's slot because they will not be given the option.

To refresh, revisit the page or reload.
